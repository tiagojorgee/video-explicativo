# CHANGELOG

Todas as mudanças importantes neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
e este projeto segue [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-05

### Added
- ✨ Primeira versão pública do Video Explicativo Generator
- 🎬 Interface web para gerar vídeos a partir de perguntas
- 🤖 Integração com GPT-4 para geração de roteiros
- 🎤 Text-to-Speech com suporte a múltiplos idiomas
- 🎨 Geração automática de slides visuais
- 📹 Compilação de vídeos em HD (1280x720)
- 🐳 Docker Compose para fácil deployment
- 📖 Documentação completa
- 🔌 API RESTful com Swagger
- ⚡ Processamento assíncrono em background

### Components
- Backend FastAPI com arquitetura modular
- Frontend React com TypeScript
- Serviços de IA, TTS, Imagem e Vídeo
- Sistema de rotas e modelos Pydantic

### Documentation
- README principal
- Backend README
- Frontend README
- Quick Start Guide
- Setup scripts (Linux/macOS e Windows)
- Checklist de instalação
- Docker commands reference
- Contributing guidelines
